# Bing Translation For Python

## 如何安装：

    $ pip install bing-translation-for-python

## 快速上手：
[完整文档][1]
### 示例：
这个示例演示了如何将任意一种语言转换为英文
#### `hello.py:`

    from bing_translation_for_python import Translator
    print(Translator('en').translator('你好'))
#### 运行 `hello.py`
    in  $ python ./hello.py
    out $ hello.
